修改页面鼠标样式

Apr 2nd, 2011 | Comments

最近在疯狂的整理电脑文件，一不小心，翻到了曾经的那个精美的鼠标集。

话说明儿个清明小长假，提前的说每个淫都要开心的度过，别整太累，虽说不是神马喜庆的日子…

下面转入正题：

  1. 应用于计算机中 下载此文件，解压后右键.ini文件，选择安装，然后到设置鼠标样式就好。XP位于控制面板–鼠标 Win7桌面右键，个性化–（左列菜单）更改鼠标指针。喜欢就拿去，别客气！

  2. 应用于网页上 最常见的就是QZone了。CSS中修改鼠标样式的属性为cursor。先研读一下属性：

    url 需使用的自定义光标的 URL。 注释：请在此列表的末端始终定义一种普通的光标，以防没有由 URL 定义的可用光标。 default 默认光标（通常是一个箭头） auto 默认。浏览器设置的光标。 crosshair 光标呈现为十字线。 pointer 光标呈现为指示链接的指针（一只手） move 此光标指示某对象可被移动。 e-resize 此光标指示矩形框的边缘可被向右（东）移动。 ne-resize 此光标指示矩形框的边缘可被向上及向右移动（北/东）。 nw-resize 此光标指示矩形框的边缘可被向上及向左移动（北/西）。 n-resize 此光标指示矩形框的边缘可被向上（北）移动。 se-resize 此光标指示矩形框的边缘可被向下及向右移动（南/东）。 sw-resize 此光标指示矩形框的边缘可被向下及向左移动（南/西）。 s-resize 此光标指示矩形框的边缘可被向下移动（北/西）。 w-resize 此光标指示矩形框的边缘可被向左移动（西）。 text 此光标指示文本。 wait 此光标指示程序正忙（通常是一只表或沙漏）。 help 此光标指示可用的帮助（通常是一个问号或一个气球）。

我们需要的就是cursor：url这项。

例如：

设置整页默认鼠标样式

html{cursor:url("images/Eyeshot/Cursor.cur"), default;}

设置应用于a属性的鼠标样式

a:hover{cursor:url("images/Eyeshot/Hand.cur"),pointer;}

至于cursor的其他属性，各位同学自行研究下。
