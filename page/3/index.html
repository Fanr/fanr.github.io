jQuery网页背景滚动代码

Feb 24th, 2011 | Comments

年前@曾哥说要主题，想了想，再做一个吧，要功能超强，体验超棒的那种…

回归主题，此功能需要jQuery的支持

代码如下：

<script type="text/javascript">// <![CDATA[
    var scrollSpeed = 70;
    var step = 1;
    var current = 0;
    var imageWidth = 1444;
    var headerWidth = 800;
    var restartPosition = -(imageWidth - headerWidth);
    function scrollBg(){
    current -= step;
    if (current == restartPosition){
    current = 0;
    }
    $('body').css("background-position","50%"+current+"px");
    }
    var init = setInterval("scrollBg()", scrollSpeed);
    // ]]>
</script>
